  1. Markdown

¿Qué es Markdown?

Markdown es un lenguaje de marcado extremadamente fácil de aprender, fácil de usar y eficaz para crear y formatear todo tipo de documentos.

Conceptualizando el Markdown

¿Empezando en Markdown? No se preocupe, eche un vistazo a nuestra guía de introducción para descubrir cómo usar el MD, fue diseñado teniendo en cuenta la practicidad y la velocidad de aprendizaje.

¿Cómo lo haces?

Seamos realistas, mismo con suficiente experiencia en cualquier lenguaje, ya sea programación o marcado, siempre existe esa duda con respecto a la sintaxis. Nuestro tutorial de sintaxis básica lo ayuda a actualizarse en la forma Markdown de formatear documentos.

¿Sólo eso?

¿Listo para ir un poco más lejos? Con la sintaxis extendida de Markdown, puede crear elementos avanzados para sus documentos, como tablas, bloques de código y links automáticos.

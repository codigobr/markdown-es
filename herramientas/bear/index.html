Bear

Bear es una aplicación simple para tomar notas para dispositivos macOS y iOS.

  1. Markdown
  3. Editores de Markdown
  5. Bear

Introducción

Bear es una aplicación muy fácil de usar, es simple, ágil y rápida, además de cumplir con los requisitos esenciales para la edición de documentos usando Markdown.

Bear permite el uso de tags, sincronización de archivos y conversión de documentos.

Pero no se deje engañar por su simplicidad, con Bear puede crear tanto documentos simples como ensayos más complejos.

Visite el sitio web oficial de Bear haciendo clic aquí.

Usando Markdown en Bear

Utilice el menú de preferencias para habilitar la compatibilidad con Markdown en la aplicación.

Con el editor en tiempo real, Bear muestra el resultado de su marcado Markdown mientras escribe su documento.

Si recién está comenzando con Markdown, ver los cambios en tiempo real lo ayudará a acostumbrarse al lenguaje.

Habilitación del soporte Markdown

Primero abra la ventana de preferencias Preferences, vaya a la pestaña General y seleccione la opción Markdown compatibility mode.

Debe hacer esto, habilitar la compatibilidad con Markdown, para cada instancia de Bear en todos sus dispositivos.

Compatibilidad de Bear con Markdown

Los elementos enumerados en Tabla 1.1 son compatibles con la aplicación Bear.

Elemento Soporte Información
Títulos Parcial Bear no proporciona soporte para la creación de títulos usando sintaxis alternativa.
Párrafos Sí
Saltos de línea Parcial No es posible romper la línea usando el espacio en blanco a la derecha del contenido, sin embargo, usando la tecla Enter puede insertar una nueva línea.
Negrita Sí
Itálico Sí
Blockquote Parcial Blockquotes anidado no son compatibles.
Listas ordenadas Sí
Listas desordenadas Sí
Código Sí
Lineas horizontales Sí
Links Sí Puede crear un vínculo colocando el nombre de la nota entre dos corchetes (ejemplo, [[nombre-nota]]).
Imagenes No La sintaxis Markdown no es compatible, pero aún puede arrastrar y soltar imágenes en una anotación.
Tablas No
Bloques de código cercados Sí También se admite el resaltado de sintaxis.
Notas al pie No
ID de título Parcial La generación de la identificación del título es automática en Bear, la personalización aún no es posible.
Listas de definiciones No
Tachado Sí
Listas de tareas Sí
Emoji (copiar y pegar) Sí
Emoji (códigos de acceso) Sí
Link automático de URL Sí
Desactivar link automático de URL Sí
HTML Parcial En el editor Bear, HTML no se representa, pero aparece correctamente después de la exportación.

Docusaurus

Docusaurus es una herramienta desarrollada por Facebook que te permite crear sitios web estáticos basados ​​en documentos Markdown.

  1. Markdown
  3. Editores de Markdown

Visión general

Docusaurus es un programa poderoso, fácil de usar y muy popular entre las organizaciones.

Al crear Docusaurus Node.js fue el pilar de desarrollo elegido por el equipo de programadores.

Los usuarios de la herramienta suelen utilizar Docusaurus para crear documentación online para varios productos, desde programas hasta frameworks.

El procesador Markdown que utiliza la aplicación es remarkable.

El sitio web oficial de Docusaurus se encuentra en https://docusaurus.io/.

Zettlr

Zettlr é um aplicativo Markdown gratuito e de código aberto desenvolvido para redação acadêmica.

Visão geral

Ele fornece muitas ferramentas poderosas para ajudá-lo a escrever textos acadêmicos mais facilmente.

O objetivo declarado do aplicativo é simples: “Permitir que pesquisadores de artes e humanas, por exemplo, pessoas sem nenhum conhecimento de programação, se libertem de programas que custam centenas de reais. Abrir caminho para uma era de código aberto. Competir seriamente com processadores de texto.”

Site: https://www.zettlr.com/

Markdown no Zettlr

O Zettlr implementa uma mistura de diferentes dialetos do Markdown. O próprio editor destaca apenas o Markdown com sabor do GitHub GitHub Flavored Markdown, além de alguns elementos extras que estendem a sintaxe do Markdown com os elementos Zettelkasten. Você também pode adicionar comandos LaTeX.
